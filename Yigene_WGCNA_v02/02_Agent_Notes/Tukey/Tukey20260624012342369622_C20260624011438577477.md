## Attacks on Strategy A+B

### Sample size risk
- Atrium-only: 23 samples total (7 Sk, 8 Sm, 8 Rn). WGCNA recommended minimum is 15-20. Feasible but weak.
- If atrium modules are unstable, the convergent claim based on atrium is unreliable.
- Diagnostic: check scale-free fit R2 and module sizes for atrium network.

### Batch confounding
- batch1=59, batch2=12. Need to verify batch is not confounded with species.
- Diagnostic: table(species, batch) before WGCNA. If one species is all batch2, results are confounded.
- FIX: include batch in voom design formula (already done: ~0+deg_group+sex+batch).

### Power selection artifact
- Round 1: auto pickSoftThreshold selected power=1, forced to 4 for better modules.
- power=1 means near-linear adjacency; modules may reflect expression magnitude rather than co-expression.
- Diagnostic: run sensitivity analysis at power=1, 4, 6 and compare module assignments.
- If modules are power-dependent, the convergent module claim is fragile.

### Unsigned network limitation
- Unsigned network merges positively and negatively correlated genes into same module.
- A convergent module might contain both up and down genes, obscuring direction.
- Diagnostic: check signed network as sensitivity; report if results change.

### Module preservation interpretation
- modulePreservation Zsummary>10 = strong, 2-10 = weak. 
- With n=7-8 per species in atrium, preservation statistics are underpowered.
- Do not claim convergence based on preservation alone; require gene set overlap corroboration.
