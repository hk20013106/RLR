# Falsification Note: WGCNA all-sample network

## Potential artifacts and weaknesses

1. Power selection: auto pickSoftThreshold selected power=1, but it was forced to 4 for better module structure.
   - At power=1, the network may be too connected (near-full graph).
   - At power=4, scale-free R2 is only 0.095 (well below 0.85 threshold).
   - This means the network is NOT scale-free at power=4. Module structure may be driven by technical variance rather than biological co-expression.
   - Risk: the 5 modules may not be biologically meaningful co-expression modules.

2. Unsigned network: signed network gave 0 modules at power=8, so unsigned was used.
   - Unsigned networks merge positive and negative correlations, which can create artificial modules from anti-correlated genes.
   - This is a known limitation; results should be interpreted with caution.

3. Only 3 species (Rn, Sk, Sm), 71 samples. Species is almost perfectly confounded with module assignment (turquoise r=-0.96).
   - This could mean the modules simply capture species-specific expression batch effects rather than conserved co-expression.
   - Without within-species replication of the biological signal, species-driven modules are hard to distinguish from batch effects.

4. green module (179 genes) is new at power=4. It did not exist at power=1 or in signed networks.
   - Small module at non-standard power may be unstable. Needs sensitivity analysis.

5. No independent validation: results are from a single network build. No bootstrap or stability check.

## Recommendations
- Run sensitivity analysis: try power=6, power=8, compare module overlap.
- Run signed network with lower mergeCutHeight to check if any modules survive.
- Prioritize GO/KEGG enrichment: if modules are biologically coherent, the power selection concern is mitigated.
