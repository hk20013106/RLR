# Idea Note — C20260623T175145

WGCNA is being performed to identify co-expression modules in the cross-species heart RNA-seq dataset (rat/bat/shrew x atrium/ventricle). The core question is which gene modules are associated with species and chamber differences, and whether ECM/collagen, mitochondrial/OXPHOS, sarcomere, immune, and fatty acid metabolism modules recapitulate or extend the DEG/ORA/GSEA findings. Three networks (all-sample, atrium-only, ventricle-only) allow chamber-stratified module discovery. Input is length-scaled counts (Strategy B corrected A) edgeR TMM voom logCPM, NOT raw counts and NOT species-removed expression.
