# Layer 8.5 — Literature Verification

- **Phase:** 4 — Result Review Loop
- **Owner:** Curie｜Evidence Auditor
- **Status effect:** none

## Purpose

Retrieve real-world literature to verify and cross-reference actual results from L7/L8, grounding the subsequent falsification (L9a) and biological interpretation (L9b) in published evidence.

## Entry condition

L8 evidence audit is complete; L7 execution results exist.

## Activities

0. **Pre-research (v0.4.5, triggered automatically):** L8.5 is in `PRE_RESEARCH_MAP`
   (`type: literature_verification`). Before this node, run
   `pre-research --node L8.5` — the prompt is grounded in the ACTUAL L7 `key_results`
   + L8 `evidence_level`/`evidence_verified`. The real PubMed/EuropePMC search runs
   there, results are written to `02_Agent_Notes/_pre_research/L8.5_research.md` and
   **embedded into this node's context** by `assemble-context`. (run_loop's
   `ensure_pre_research` triggers it in headless mode.)
1. Using the injected pre-research papers (+ any additional `academic-research-suite`
   search), compare each paper's findings with this study's L7/L8 results.
2. Extract PMIDs, titles, abstracts; mark each as confirms / contradicts / extends.
3. Add papers to the literature DB (`manage_literature_db.py`).
4. Output the structured delta JSON.

## Exit condition

L8.5 delta emitted. Route to **L9 (Feynman + Darwin)**.
