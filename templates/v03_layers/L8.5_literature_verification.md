# Layer 8.5 — Literature Verification

- **Phase:** 4 — Result Review Loop
- **Owner:** Curie｜Evidence Auditor
- **Status effect:** none

## Purpose

Retrieve real-world literature to verify and cross-reference actual results from L7/L8, grounding the subsequent falsification (L9a) and biological interpretation (L9b) in published evidence.

## Entry condition

L8 evidence audit is complete; L7 execution results exist.

## Activities

1. Formulate precise literature search keywords based on actual findings from L7 and L8 (e.g., specific genes like Col6a1, specific heart rate phenotypes, bat/shrew cardiac traits).
2. Call `academic-research-suite` (or search PubMed/EuropePMC) to search for real papers.
3. Extract PMIDs, titles, abstracts, and compare their findings with this study's actual findings.
4. Output structured delta JSON.

## Exit condition

L8.5 delta emitted. Route to **L9 (Feynman + Darwin)**.
