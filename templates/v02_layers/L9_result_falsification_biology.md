# Layer 9 — Result Falsification + Biology

- **Phase:** 4 — Result Review Loop
- **Owners:** Feynman｜Reality Checker (falsification) + Darwin｜Evolutionary Biologist (biology)
- **Status effect:** none (Oppenheimer decides at L10)

## Purpose

Attack the completed result for hidden confounders and false positives
(Feynman), then interpret it biologically without overreaching (Darwin).

## Entry condition

Curie's `support_level.md` exists.

## Activities — Feynman

1. Attack the result: confounders, collinearity, pseudoreplication, false positives.
2. Request concrete diagnostics (partial correlation, permutation null, etc.),
   routed to Turing via Oppenheimer.

## Activities — Darwin

1. Interpret modules/pathways/tissues/species patterns in context.
2. Separate correlation, mechanism, adaptation, speculation.
3. Mark plausible-but-unproven biology explicitly.

## Required outputs

- `result_falsification.md` (Feynman)
- `diagnostic_tests_requested.md` (Feynman)
- `biology_interpretation.md` (Darwin)
- `literature_support.md` (Darwin)
- `mechanism_risk.md` (Darwin)

## Exit condition

Falsification + biology recorded. Route to **L10 (Jobs → Oppenheimer)**.

## Forbidden

Feynman: no veto, no vague criticism. Darwin: no causation from correlation, no
adaptation claim without adequate evidence, no enrichment-as-mechanism, no
replacing the evidence audit.
