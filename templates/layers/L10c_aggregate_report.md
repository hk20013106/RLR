# L10c — Aggregate Report

## Purpose

Compile an accurate, traceable report of the completed loop from its recorded artifacts.

## Reasoning boundary

- Summarize provenance, decisions, evidence, limitations, and unresolved items without creating new analysis.
- Preserve links between claims and the artifacts that support them.
- Treat missing artifacts and failed steps as reportable facts.

## Handoff

Produce the final aggregate record only. Do not revise decisions, execute work, or infer unrecorded findings.

## Full-mode role

The current dynamic contract is authoritative for inputs, output schema, state changes, and runtime commands.
